It highlights sections such as Home, About, Experience, Projects, and Contact.

## Overview

The Personal Portfolio includes:

- Landing/Home section
- About Me section
- Experience and skills
- Projects/Featured Works
- Contact Me section

The website is fully frontend-based and does not require a backend.

## Features

- Fully responsive layout
