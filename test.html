HOW I WORK

00
Framework

Principles that guide my development. Not rigid rules, but a living compass. Every line of code, every layout decision is shaped by these beliefs. They keep me grounded and moving forward.

Clean. Human. Performant. Transparent. Ever‑evolving.
This is how I build for the web.

01
CLEAN CODE

I focus on writing readable, structured, and maintainable code that is easy to scale and improve over time. Indentation, naming, modularity — these aren't afterthoughts, they're core to resilience. Whether it's CSS or JavaScript, I favor clarity over cleverness.

“Always code as if the person who ends up maintaining your code is a violent psychopath who knows where you live.” — I embrace that spirit with kindness.

02
USER-FIRST

I design and build with the user in mind, making sure layouts are intuitive, accessible, and responsive on all devices. Every component should feel natural — from touch targets to color contrast, from keyboard navigation to reduced motion preferences. Empathy is a requirement.

No dark patterns. Just honest, frictionless interfaces.

03
PERFORMANCE

I pay attention to performance by optimizing assets, minimizing unnecessary code, and ensuring smooth interactions. Fast load times, efficient reflows, and lightweight builds are not "nice to have" — they're part of the user experience.

Lazy loading, code splitting, critical CSS, modern formats. I keep an eye on the metrics that matter.

04
COMMUNICATION

I value clarity and transparency — sharing progress, explaining decisions, and aligning solutions with real requirements. Whether it's code comments, pull requests, or client conversations, I aim to be direct and respectful.

Assumptions are dangerous. Questions are welcome. Documentation is a gift to my future self.

05
LEARNING

I actively improve my skills by exploring new tools, techniques, and best practices in modern web development. The landscape shifts — I shift with it. I experiment, break things, read specs, and share insights.

From container queries to AI‑assisted workflows, I stay curious and humble.
